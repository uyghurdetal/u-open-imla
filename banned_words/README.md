#### 如何过滤禁用词？

Python 代码示例如下：

```python
from shirkhan import md5

benned_word_list = ["x", "y", "z"],
target_word_list = ["a", "b", "c"]
for ww in target_word_list:
    if md5(ww) in benned_word_list:
        print("禁用词")
```
