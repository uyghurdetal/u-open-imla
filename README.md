# 常用 shell 命令

统计 corpus 中各个文件 和行数

```shell
 ls ./corpus/sources/**/*.txt|xargs -n 1 -I {} wc -l {}
```

统计总 corpus 行数

```shell
ls ./corpus/sources/**/*.txt|xargs -n 1 -I {} wc -l {}|awk -F " "  '{sum += $1} END {print sum}'
```
